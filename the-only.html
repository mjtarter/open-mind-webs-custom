The Only

  • Web Development
  • Web Design
  • Content Development
  • Database Management

The Only is a sports exhibition hall that consists of thousands of historical sports pieces and memorabilia. For this project the client needed an organized database to display all of their inventory. We photographed each individual item, and placed numbered id stickers on them. Then we created the database and website to the right. It consists of search criteria including key word search to find specific items. It also allows the manager to add and delete items from the database.
