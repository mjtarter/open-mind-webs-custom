MyCampus Island

  • Web Development
  • Web Design
  • Responsive Design
  • Content Development
  • Database Management

MyCampus Island is a web based student housing app for college students in the Bloomington, Indiana area. This is a large site that takes advantage of an extensive back-end database to provide in-depth individual pages for all 12 dorms, as well as over 1,000 properties in Bloomington. This is a fully responsive website equipped with search criteria that allows users to find specific housing properties in Bloomington.
